#### checkout already branch
```
git checkout XXXX 
```

#### create new brach
```
git checkout -b xxxx
```

#### first push to remote
```
git push -u origin xxxx
```

#### create branch from remote 
```
git checkout -b XXXX origin/XXX
```

#### back to latest committed
```
git reset --hard HEAD
git reset COMMIT_ID //git log
```

#### show markdown tab
```
ctrl+shift+v
```
